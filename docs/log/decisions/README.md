# 决策记录索引

本目录存储架构决策记录（Architecture Decision Records, ADR）。

---

## 什么是 ADR？

架构决策记录用于记录重要的技术决策，包括：
- 决策背景
- 考虑的选项
- 最终决定
- 权衡分析

---

## 如何创建新的决策记录

1. 复制 `TEMPLATE.md` 到 `[decision-name].md`
2. 填写各个部分
3. 在本文件底部添加索引条目
