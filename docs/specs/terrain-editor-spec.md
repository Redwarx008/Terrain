# Stride 地形编辑器 - 产品规格文档

## 产品愿景

**实时 3D 预览的笔刷式地形编辑器** — 为非公开项目提供所见即所得的高度和材质编辑体验。

Stride 地形编辑器是一个独立应用程序（非 Stride Studio 插件），支持高度图编辑和材质槽绘制。开发者通过直观的笔刷系统实时编辑地形，最终导出为 `.terrain` 格式或 PNG 序列，供 Stride 游戏引擎运行时使用。

---

## 目标用户

本编辑器为非公开项目定制开发，满足项目特定的地形编辑需求。

---

## 核心价值主张

### 所见即所得

在 3D 视口中直接编辑地形，每一次笔刷操作都即时反映在渲染结果中。开发者无需切换视图或等待预览更新，编辑体验流畅自然。

### 快速迭代

- **实时反馈**：笔刷操作即时生效，无延迟感
- **撤销重做**：支持多步撤销，鼓励尝试不同方案
- **快速保存**：项目保存和导出操作简单快捷

### 引擎原生

输出格式直接兼容 Stride 运行时：
- `.terrain` 二进制格式，支持流式加载和 LOD
- PNG 高度图序列，便于版本控制和外部处理

## 功能模块

### 模块一：高度编辑

> 优先级：最高  
> 状态：部分实现

开发者最基础的需求是创建有趣的地形形状。高度编辑模块提供直观的"雕刻"体验。

#### 编辑工具

| 工具 | 功能描述 |
|------|----------|
| **隆起 (Raise)** | 向上推高地形高度 |
| **降低 (Lower)** | 向下压低地形高度 |
| **平滑 (Smooth)** | 软化地形起伏，减少锯齿感 |
| **整平 (Flatten)** | 将地形拉平到参考高度 |

#### 笔刷参数

| 参数 | 范围 | 说明 |
|------|------|------|
| 大小 | 1 - 200 | 笔刷半径（世界单位） |
| 强度 | 0 - 1 | 每次操作的影响程度 |
| 衰减 | 0 - 1 | 笔刷边缘的柔和度 |

#### 笔刷形状

- **圆形**（已实现）- 标准圆形笔刷
- **方形**（计划中）- 适用于创建平台、台阶
- **平滑**（计划中）- 边缘更柔和的圆形
- **噪点**（计划中）- 带随机扰动的形状，用于自然效果

#### 撤销/重做

- 支持多步撤销（计划中）
- 快捷键支持：Ctrl+Z / Ctrl+Y
- 撤销历史可配置上限

---

### 模块二：材质绘制

> 优先级：高  
> 状态：UI 已完成，功能待实现

地形不仅要有形状，还要有颜色和质感。材质绘制模块让开发者为地形"上色"。

#### 材质槽位系统

- 支持 256 个材质槽位（0-255）
- 每个槽位对应一种材质纹理
- 通过 SplatMap 记录材质分布

#### 绘制工具

| 工具 | 功能描述 |
|------|----------|
| **绘制 (Paint)** | 在地形上绘制选中材质 |
| **擦除 (Erase)** | 移除已绘制的材质，露出底层 |

#### 材质管理

- 导入纹理作为材质源
- 设置纹理平铺比例
- 材质预览和快速切换

---

### 模块三：植被放置

> 优先级：中  
> 状态：UI 已完成，功能待实现

为地形添加树木、岩石、草丛等装饰元素，让场景更加生动。

#### 放置工具

| 工具 | 功能描述 |
|------|----------|
| **放置 (Place)** | 在地形上放置植被实例 |
| **移除 (Remove)** | 删除已放置的植被实例 |

#### 植被管理

- 导入植被模型（树、岩石等）
- 设置放置密度和随机范围
- 朝向和缩放的随机化

## 用户体验原则

### 直观性

- 笔刷操作符合直觉：左键绘制，右键旋转视角
- 工具栏图标清晰易懂
- 参数滑块实时预览效果

### 响应性

- 笔刷操作延迟低于 16ms（60 FPS）
- 视口渲染流畅，无卡顿
- 大地形分页处理，避免内存溢出

### 学习成本低

- 无需阅读手册即可上手基本操作
- 工具提示提供即时帮助
- 界面布局简洁，功能分区明确

## 输出格式

### .terrain 二进制格式

专为此项目设计的地形数据格式，支持 Stride 运行时的高效加载。

**特点：**
- 内嵌 Mipmap 层级，支持 LOD
- 虚拟纹理布局，支持流式加载
- 四叉树空间索引，快速可见性判断

**适用场景：**
- Stride 游戏运行时
- 大型开放世界地形

### PNG 高度图序列

标准图像格式，便于版本控制和外部工具处理。

**特点：**
- L16 灰度格式（16-bit 精度）
- 分块导出，支持超大地形
- 人类可读，便于 Git 管理

**适用场景：**
- 版本控制
- 与其他工具互操作
- 备份和归档

## 附录：术语表

| 术语 | 英文 | 解释 |
|------|------|------|
| 高度图 | Heightmap | 记录地形高度的灰度图像，亮度代表高度 |
| SplatMap | SplatMap | 记录材质分布的纹理，每个像素存储材质权重 |
| LOD | Level of Detail | 细节层次，远处使用简化模型 |
| 虚拟纹理 | Virtual Texture | 大纹理的分页加载技术，按需加载可见部分 |
| 四叉树 | QuadTree | 空间划分数据结构，用于 LOD 选择和可见性判断 |
| 植被 | Foliage | 地形上的装饰对象（树木、岩石、草丛等） |
| 材质槽位 | MaterialSlot | 存储单一材质配置的槽位，支持 256 个 |
| 衰减 | Falloff | 笔刷边缘的柔和度，控制影响范围渐变 |
| 类型索引 | TypeIndex | 植被类型在类型列表中的索引 |
